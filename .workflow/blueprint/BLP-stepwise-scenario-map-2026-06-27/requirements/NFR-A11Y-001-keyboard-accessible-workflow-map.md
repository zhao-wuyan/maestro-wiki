# NFR-A11Y-001: Keyboard Accessible Workflow Map

The workflow map MUST remain keyboard accessible as scenario steps and continuation routes are added.

## Acceptance Criteria

- Interactive nodes and route actions MUST be reachable by keyboard.
- Current focus SHOULD be visually apparent.
- Detail panel updates SHOULD be announced or structured for assistive technologies.
