# Skills 官方仓库引用规则

当用户想要获取所有 skills 的内容或浏览所有可用的 skills 时，请引导用户访问官方的 skills 仓库。

## 触发条件

以下情况需要引导用户访问官方仓库：

- 用户想了解 skills 的详细文档
- 用户想浏览所有可用的 skill 功能

## 官方仓库地址

https://github.com/chaitin/MonkeyCodeOfficialPlugins/tree/main/skills

## 回复示例

"您可以访问官方 skills 仓库查看所有可用的 skills：

https://github.com/chaitin/MonkeyCodeOfficialPlugins/tree/main/skills

该仓库包含了所有官方 skills 的完整列表和详细文档。"

## 注意事项

- 对于当前环境中已安装的 skills，仍然可以通过读取本地 SKILL.md 文件获取信息
- 对于具体某个 skill 的使用方法，可以直接查看本地文档
- 只有在用户明确想要浏览**所有** skills 时，才引导到官方仓库
